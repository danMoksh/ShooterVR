## About The Project

ShooterVR is a stationary "wave shooter" where the player uses their head to aim and a single button press to fire a laser. The game environment is a simple terrain with a 360-degree skybox.

The objective is to find and "destroy" 5 spinning cubes (enemies). Once all 5 enemies are destroyed, the game loads a "Project Submitted" scene. This project serves as a practical demonstration of fundamental VR mechanics in Unity.

## Gameplay Features

* **VR Gaze-Based Aiming:** Uses the Google VR SDK for head tracking and aiming.
* **Raycast Shooting:** Fires an instantaneous laser using a `Raycast` and visualizes it with a `LineRenderer`.
* **Simple Enemy AI:** Enemies are spinning cubes that are "destroyed" (set inactive) when hit.
* **Win Condition:** The game tracks the number of destroyed enemies and loads a "Win" screen upon defeating all 5.
* **3D Audio:** Implements shooting sounds and background music.

## ğŸš€ Getting Started

There are two ways to run this project: you can play the pre-compiled APK on an Android phone (Running Android 13), or you can build it from the source code.

### To Play (APK)

You can download the latest `ShooterVR.apk` from the **[Releases](https://github.com/danMoksh/ShooterVR/releases)** page of this repository.

1.  Download the `.apk` file to your Android phone.
2.  You may need to "Allow installation from unknown sources" in your phone's security settings.
3.  Install and run the app.
4.  Place your phone into a Google Cardboard (or similar) headset to play.

### To Build from Source

To open this project in the Unity Editor:

1.  Clone this repository: `git clone https://github.com/danMoksh/ShooterVR.git`
2.  Open **Unity Hub** and click "Add project".
3.  Navigate to and select the cloned `ShooterVR` folder.
4.  Unity will open the project and automatically build the `Library` folder.
5.  In the Project window, navigate to the `_Scenes` folder and open the `MainMenu` scene to begin.
